# Students Table - React.js

A simple **frontend-only Students Table** application built with **React.js**.  
This app allows you to **add, edit, delete, and download student data** entirely on the frontend using in-memory state.

---

## **Features**

- **Student List Table**  
  - Columns: Name, Email, Age, Actions (Edit/Delete)
- **Add Student Form**  
  - All fields mandatory  
  - Email validation
- **Edit Student**  
  - Pre-filled form with validations
- **Delete Student**  
  - Confirmation dialog before deletion
- **Simulated Loading State**  
  - Shows a loading indicator while processing actions
- **Excel Download**  
  - Download full or filtered student data

---

## **Tech Stack**

- **React.js** (Functional Components & Hooks)  
- **JavaScript ES6+**  
- **HTML / CSS**  
- **xlsx** library for Excel download
